动态内存

关于对动态内存的一些见解

动态内存，按我的理解就是：可以随用户的需要，
数组开多大就开多大，不用预设开100000000000…，
然后就炸了…………
所以，这时候需要动态内存！
（当然，也要取决于你的电脑的内存大小……）

1. 申请一维数组

//申请一维数组的动态内存： 
int One_dimensional_Array() {
    int size;        //申请的空间大小； 
    cout<<"请输入申请空间的大小：";
    cin>>size;
    int *array=new int [size];        //为int型的变量请求空间； 
    if(!(array=new int [size])) {       //判断是否成功分配内存； 
        cout<<"Error: out of memory!"<<endl;
        exit(1);      //正常为0，错误为1||-1，找不到文件为2； 
    }
    cout<<"请输入数据："<<endl; 
    for(int i=0;i<size;i++) cin>>array[i];      //两种读入数据相同，一个是用数组 
    for(int i=0;i<size;i++) cout<<*(array+i);  //这里的是用指针
    delete [] array;                          //释放内存； 
    return 0;
}

2. 申请二维数组

//申请二维数组的动态内存：
int Two_dimensional_Array() {
//在菜鸟教程上的解释：假定数组第一维长度为 m， 第二维长度为 n
//而我的理解是： 其实就是类同于二维数组的x,y轴
//（当然，在计算机中的储存一样是跟一维数组一样……） 
    int size_x,size_y;
    cout<<"请输入x轴与y轴的大小：";  //其实也就是一维的长度与二维的长度； 
    cin>>size_x>>size_y;
    int **array;      //指针的指针； 
    array=new int *[size_x]; 
//其实就是size_x * size_y 也就是：array[size_x][size_y]
//也就是变量的个数的定义等同于二维数组；  
    for(int i=0;i<size_x;i++) array[i]=new int [size_y];  //申请完成
    cout<<"请输入数据："; 
    for(int i=0;i<size_x;i++)
    for(int j=0;j<size_y;j++)
    cin>>array[i][j]; 
    for(int i=0;i<size_x;i++) 
    for(int j=0;j<size_y;j++) 
    cout<<array[i][j]; 
    for(int i=0;i<size_x;i++) delete [] array[i];   //先释放二维的；
    delete [] array;     //再释放一维的；
    return 0; 
}

3. 申请三维数组

//申请三维数组的动态内存：
int Three_dimensional_Array() {
    int size_x=0,size_y=0,size_z=0;
    cout<<"请输入x,y,z轴的大小：";
    cin>>size_x>>size_y>>size_z;
    int ***array;  //指针的指针的指针
    array=new int **[size_x];
//一层一层的申请空间： 
    for(int i=0;i<size_x;i++) {
        array[i]=new int *[size_y];
        for(int j=0;j<size_y;j++) array[i][j]=new int [size_z];
    } 
    cout<<"请输入数据：";
    for(int x=0;x<size_x;x++)
        for(int y=0;y<size_y;y++)
            for(int z=0;z<size_z;z++)
                cin>>array[x][y][z];
    for(int x=0;x<size_x;x++)
        for(int y=0;y<size_y;y++)
            for(int z=0;z<size_z;z++)
            cout<<array[x][y][z];
//层层释放：
    for(int i=0;i<size_x;i++) {
        for(int j=0;j<size_y;j++) delete array[i][j];
    } 
    delete [] array;
    return 0;
}

1. 创建

//创建链表：
node *createlist(int total) {
    int num=0;
    if(total<=0) return NULL;
    r=new node;                //申请动态内存， 
    h=r;                   //头链表一般不用，但它是一个链表开头的地址，可以通过访问它以访问其他表；
    for(int i=0;i<total;i++) {
        cin>>num;            //读入数据； 
        p=new node;         //每一次循环都申请一次动态内存，等于下一个链表； 
        p->data=num;        //存入数据； 
        r->next=p;            // 把下一个表的地址存起来； 
        r=p;                // 尾指针指向刚刚 工作指正的地址，工作指针指向下一个表 （等于回到循环开头）；  
    } 
    r->next=NULL;           //当尾指针指向NULL（空）时，链表结束； 
    return (h);                //传回头链表的地址，以方便其他函数访问链表； 
}

node *putlist(node *h) {    //通过头链表，

}

C++的标准模板 与 stl（面向对象）

namespace(命名空间)

现在的软件多以程序庫、模块、组件拼凑而成，名称冲突问题越来越严重。Namespace就是用来解决此问题的。 Namespace将不同的标识符号集合在一个具体的名称范围内。

模板的格式

比较两个数，取最大：

template<class T> //标准格式，“T”可以任取
T max(T a[],int n) //类似于自定义函数，按我理解，class 的意义在于可自动识别变量类型
{

模板的总结

C++功能强大，为开发者提供了标准模板库，其中封装了很多实用的容器。容器可以理解成能够实现很多功能的系统函数，或者说用来存放数据的对象，开发者可以根据接口规范（调用格式）直接调用，而不用关心其内部实现的原理和具体代码，十分方便快捷。

STL包含六大组件：

容器（container）、算法（algorithm）、迭代器（iterator）、配置器（allocator）、适配器（adapter）和函数对象（functionobject）。我们在学习这些组件时，应该按其重要程度来区别学习。重要成都由大到小是：泛型容器和泛型算法（表示任何类型和对象都可以使用这些容器和算法）>>迭代器>>配置器、适配器、函数对象。

STL的各部分作用是

1）为了方便地存储数据，设计了容器；

2）为了方便地遍历、查找、替换容器中的元素，设计了算法；

3）为了方便地让容器、算法独立工作，设计了在二者之间起桥梁作用的迭代器；

4）为了统一分配和控制容器中的内存，设计了配置器；

5）为了更好地给算法传入参数，设计了函数对象；

6）为了更好地扩展STL现有的接口，设计了适配器。
